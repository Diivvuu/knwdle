# Knwdle Community Code of Conduct

We want Knwdle to be a welcoming, respectful, inclusive community.

This code applies to:
- Issues  
- PRs  
- Discussions  
- Any public interactions related to Knwdle  

---

## 1. Be Respectful
- Be kind and constructive  
- Disagree with ideas, not people  
- No personal attacks or harassment  

---

## 2. No Discrimination
We do not tolerate:
- Hate speech  
- Racism, sexism, homophobia  
- Harassment or threats  

---

## 3. Be Helpful
- Give constructive feedback  
- Assume good intentions  
- Help newcomers  

---

## 4. No Spam
- No advertising unrelated projects  
- No irrelevant self-promotion  

---

## 5. Reporting Problems
If someone violates this Code:

- Contact the maintainer privately  
- Or use GitHub reporting tools  

Actions may include warnings, temporary bans, or permanent bans.
